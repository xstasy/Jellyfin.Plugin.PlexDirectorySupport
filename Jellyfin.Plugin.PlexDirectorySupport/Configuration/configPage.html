Plex Directory Support

Settings

Token Sweep

Scans movie and series parent folders for Plex-style tokens such as {tmdb-12345}, {tvdb-12345}, and {imdb-tt1234567}.

Normal sweep

Injects missing IDs and refreshes only tokened items that have not already completed a PlexDirectorySupport refresh for the same token set.

Force sweep

Refreshes every tokened movie and series parent item, including items already handled before.
